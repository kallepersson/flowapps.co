Tab Switcher Extension

Rapidly control, filter and switch tabs
Download v1.7.5 on Chrome Web Store

It understands what you mean

Tab Switcher lets you to quickly switch to other open tabs using only the keyboard, including filtering tabs by partial name or URL match.

Interactive preview coming soon!

Designed to save time

⌘⇧K to start filtering, move with ↑↓ and press ⏎ to switch.

Commands are constantly added. Here's a list of the current ones:

Close tabs in list :close or :x
Reload tabs in list :reload or :re
Move tabs in list to their own window :detach or :win
Deduplicate tabs in list :deduplicate or :dd
Sort tabs in list alphabetically by URL :sort
